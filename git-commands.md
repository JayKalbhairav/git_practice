# Git Commands Reference

## Setup & Configuration

### git --version

**Purpose:** Shows the installed Git version.

**Example:**

```bash
git --version
```

### git config --global user.name

**Purpose:** Sets your Git username.

**Example:**

```bash
git config --global user.name "John Doe"
```

### git config --global user.email

**Purpose:** Sets your Git email address.

**Example:**

```bash
git config --global user.email "john@example.com"
```

### git config --list

**Purpose:** Displays Git configuration settings.

**Example:**

```bash
git config --list
```

---

## Basic Workflow

### git init

**Purpose:** Creates a new Git repository.

**Example:**

```bash
git init
```

### git add

**Purpose:** Adds changes to the staging area.

**Example:**

```bash
git add git-commands.md
```

### git commit

**Purpose:** Saves staged changes to repository history.

**Example:**

```bash
git commit -m "Add Git commands reference"
```

---

## Viewing Changes

### git status

**Purpose:** Shows repository status and tracked/untracked files.

**Example:**

```bash
git status
```

### git log

**Purpose:** Displays commit history.

**Example:**

```bash
git log
```
